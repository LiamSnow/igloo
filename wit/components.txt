# This file describes all components in Igloo
# 
# *Modification*:
#   1. *Immutable*: Components can never be renamed or have their type changed
#       - The only exception to this is that new variants can be added to enums at the bottom
#   2. *Append Only*: All new components must be added at the end of the file
#
# *Syntax*:
#   <Ident> <(<IglooType> | 'none')> '\n'
#   <Ident> 'enum' '{' '\n' <(<Ident> '\n')*> '}' '\n'
